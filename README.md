# story-boards

Technology used Vue, Vuex, feathersjs, mongodb, axios

Client:

npm install

npm start

Runs the app in the development mode.
Open http://localhost:8080/ to view it in the browser.

Server:

npm install

npm start

Runs the app in the development mode.
Open http://localhost:3030 to view it in the browser.

Install 'mongodb'
